# stockspec_vue
frontend for stockspec using vue
用Vue.js做股票指标的前端

## 开发环境

IDE: visual studio code

Requirements:
axios: 0.19.2,
element-ui: 2.13.1,
highcharts: 8.0.4,
highcharts-vue: 1.3.5,
moment: 2.24.0,


## 运行

```
cd stockspec

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
